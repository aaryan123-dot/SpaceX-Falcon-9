# Space-X Falcon 9
-> Collected the SpaceX Falcon 9 launch data by performing Webscrapping with the help of \textbf{BeautifulSoup}. Performed Data Wrangling and EDA with the help of \textbf{SQL}.
-> Built an interactive map with \textbf{Folium} and ran Classification algorithms (KNN, Decision Tree Classifier, SVM, Logistic Regression) and tested their accuracy.
